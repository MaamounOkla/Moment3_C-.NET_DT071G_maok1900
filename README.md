## Beskrivning
Detta repository innehåller en konsolapplikation skriven i C# .NET som ingår i Moment 3 i kursen Programmering i C#.NET.

Applikationen är en enkel gästbok uppbyggd med OOP (objektorienterad programering).

## Funktionalitet
- Visa alla gästboksinlägg
- Lägga till nya inlägg
- Ta bort inlägg via index
- Enkel meny i konsolen
- Kontroll och felhantering av tomma inmatningsfält
- Lagring av data i JSON-fil


## Programstruktur:
- Program.cs – meny och användarinmatning.
- GuestBook.cs – logik och filhantering.
- GuestbookEntry.cs – modell för gästboksinlägg.
